# Wedding Library
  
### Install and usage  
  
```
npm install -S alfa-wedding
```  

_as es6 module:_  
```
import Wedding from 'alfa-wedding';
// use Wedding constructor
```  
  
_in browser:_  
```
<script src="path/to/wedding.umd.js";
// now Wedding constructor exists in global  
// use Wedding constructor  
```
